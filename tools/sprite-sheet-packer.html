    Welcome to AtlasForge

    AtlasForge is a professional, entirely client-side tool for packing multiple images into highly optimized sprite sheets.

    Key Features

    • MaxRects Algorithm: Uses a mathematically rigorous bin-packing heuristic (Best Short Side Fit) to heavily minimize wasted texture space.
    • SVG Symbol Maps: Upload vector .svg files, and the app will generate a production-ready <symbol> map with rewritten namespace IDs to prevent DOM collisions.
    • Retina Scaling: Easily export CSS coordinates dynamically scaled down for high-DPI @2x or @3x displays.
    • 100% Private: All processing happens directly in your browser tab. No assets are ever uploaded to any external server.

    Supported Formats

    Accepts PNG, JPEG, WebP, and SVG files via batch upload or drag-and-drop.
