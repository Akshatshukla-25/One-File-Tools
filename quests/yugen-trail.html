The Expedition Field Guide

✕
Every discovered path becomes a lantern, hung on a single chain. Lanterns are only ever lit from the front — the very first one hung. New lanterns, spied down side paths, are tied on at the back of the chain. This is a queue: first in, first out.

What it actually is

Breadth‑First Search explores a graph one full "ring" of distance at a time. It visits every neighbor of the start before moving on to their neighbors, guaranteeing it reaches any node by the shortest possible number of steps, in an unweighted graph.

The code, in short

function bfs(graph, start) {
  const visited = new Set([start]);
  const queue = [start];        // the lantern chain
  const order = [];

  while (queue.length) {
    const current = queue.shift();  // unlit from the FRONT
    order.push(current);

    for (const next of graph[current]) {
      if (!visited.has(next)) {
        visited.add(next);
        queue.push(next);       // tied on at the BACK
      }
    }
  }
  return order;
}
O(V + E) time · O(V) space

Where it walks in the real world

  • Shortest hop-count routing — social‑network "degrees of separation", unweighted road maps.
  • Level‑order traversal of trees and org charts.
  • Web crawlers that fan out one link‑depth at a time.
  • Finding the nearest reachable resource in a grid, like fire spreading evenly outward.
A single rope uncoils ahead of you into the fog. You keep feeding it deeper down one path until you hit a dead end — then you pull the rope back, hand over hand, to the last fork, and try the next thread from there. This is a stack: last in, first out.

What it actually is

Depth‑First Search commits to a single path as far as it can go before backtracking. It doesn't guarantee the shortest route, but it's remarkably good at exhausting every possibility along one branch — and at remembering exactly how to retrace its steps.

The code, in short

function dfs(graph, start) {
  const visited = new Set();
  const stack = [start];         // the coiled rope
  const order = [];

  while (stack.length) {
    const current = stack.pop();  // unwound from the TOP
    if (visited.has(current)) continue;
    visited.add(current);
    order.push(current);

    for (const next of [...graph[current]].reverse()) {
      if (!visited.has(next)) stack.push(next); // coiled on TOP
    }
  }
  return order;
}
O(V + E) time · O(V) space

Where it walks in the real world

  • Maze solving and puzzle backtracking (Sudoku solvers, N‑Queens).
  • Topological sorting and dependency resolution (build systems, package managers).
  • Detecting cycles in graphs and file‑system directory traversal.
  • Generating permutations, combinations, and game‑tree exploration.
